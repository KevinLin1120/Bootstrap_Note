# Start

## Download

1. CDN

2. Download file:

    1. 官網 > Docs > Download > Compiled CSS and JS

    2. 將 css 與 js 資料夾放入專案資料夾

    3. 引入 bootstrap.min.css, bootstrap.min.js

    4. 上去 CDN 來源網址下載 poper.min.js 並放入專案資料夾, 引入
